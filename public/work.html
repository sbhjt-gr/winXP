Inferra

Inferra is a mobile application that brings large language models (LLMs) and small language models (SLMs) directly to your Android and iOS device. Cloud-based models like Claude, DeepSeek, Gemini, and ChatGPT are also supported.

Core Features:

  • Local inference through llama.cpp with support for GGUF models
  • Seamless integration with cloud-based models (OpenAI, Gemini, Anthropic, DeepSeek)
  • Vision support through multimodal models with projector files
  • Built-in camera for capturing and sending images to models
  • RAG (Retrieval-Augmented Generation) support for document understanding
  • File attachment support with built-in document extractor and OCR

Model Management:

  • Download models directly from HuggingFace
  • Cherry-picked model list optimized for edge devices
  • Import models from local storage or custom URLs
  • Built-in HTTP server with REST APIs for network access

Chat Experience:

  • Messages support editing, regeneration, and markdown rendering
  • Code blocks with clipboard copying functionality
  • Chat history management and organization
  • Real-time streaming responses for local and remote models

Technologies Used:

  • React Native with Expo
  • TypeScript
  • Firebase (authentication, Firestore)
  • llama.cpp for local inference
  • React Native ML Kit for OCR
  • WebRTC for peer-to-peer connections

WiLang

WiLang is a comprehensive video calling application built with Expo and React Native that delivers professional-grade communication features across iOS, Android, and web platforms.

Key Features:

  • Firebase Authentication with email/password and Google Sign-In
  • WebRTC-powered peer-to-peer video calls
  • Meeting rooms with unique IDs for group participation
  • One-to-one calls directly from contacts
  • Audio/video controls (mute, camera switch, end call)
  • Real-time connection status and participant management

Technical Implementation:

  • React Native with Expo SDK 53
  • TypeScript for type safety
  • React Navigation 7 with native stack and tab navigators
  • React Native Elements (RNEUI) for UI
  • WebRTC with Socket.IO signaling
  • Firebase Auth and Firestore

App Structure:

  • Calls Tab - Create instant meetings, join existing meetings
  • Contacts Tab - Access device contacts, initiate video calls
  • History Tab - View call history and past meetings
  • Settings Tab - User preferences and account management

MydenFocus

MydenFocus is a productivity Android app designed to enhance academic performance through focused study sessions, meditation, and task management. Built with modern Android development practices using Jetpack Compose.

Features:

  • Study Session Timer - Track study sessions with customizable timer
  • Meditation Timer - Built-in meditation timer with background music options
  • Task Management - Schedule and track academic tasks with notifications
  • Subject Management - Organize study sessions by subjects
  • Progress Tracking - Monitor study hours and meditation sessions

Tech Stack:

  • UI: Jetpack Compose with Material 3
  • Architecture: MVVM with Clean Architecture
  • Dependency Injection: Hilt
  • Navigation: Compose Destinations
  • Background Processing: Kotlin Coroutines
  • Local Storage: Room Database
  • Services: Foreground Services for timers
